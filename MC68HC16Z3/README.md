# MC68HC16 (CPU16) processor module

Ghidra SLEIGH processor module for the Motorola/Freescale MC68HC16 (CPU16) core, tuned for the
Chrysler JTEC+ PCM (MC68HC16Z3). Drop this `MC68HC16Z3/` folder into
`<GhidraInstallDir>/Ghidra/Processors/` and select language **68HC16** on import.

See the [repository README](../README.md) for installation, the CPU16 banking model, JTEC+-specific
assumptions, and licensing (Apache-2.0).
